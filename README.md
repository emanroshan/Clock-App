# Clock App

This project aims to create a lightweight version of the Android Clock app, designed to help users manage time with ease. It includes essential features like a digital clock, alarm, stopwatch, and timer. The app prioritizes simplicity and usability with a clean interface that mirrors the original Android Clock app. The focus is on delivering a reliable and responsive app that provides users with everything they need to manage their time effectively, without overwhelming them with complex options.

## Features

### 1. Digital Clock
**Description**:  
A simple view that displays the current time in a clean, minimalist Android-style digital clock.

**Details**:
- Uses the phone’s system time and updates in real-time.
- Users can switch between 12-hour and 24-hour formats depending on their preferences.
- The UI is clear and easy to read, closely resembling the original Android Clock app.

---

### 2. Alarm
**Description**:  
A robust alarm system that allows users to easily set, edit, and delete alarms.

**Details**:
- Users can set alarms for specific times.
- Choose alarm tones and enable options like snooze.
- Supports recurring alarms (daily or weekly schedules).
- Deleting alarms is quick and simple.

---

### 3. Stopwatch
**Description**:  
A fully functional stopwatch to track time, perfect for tasks that require precise timing.

**Details**:
- Users can start, pause, and reset the stopwatch.
- Track lap times for activities that require split times.
- The interface displays hours, minutes, seconds, and milliseconds for accuracy.

---

### 4. Timer
**Description**:  
A countdown timer to help users time specific tasks.

**Details**:
- Users can set the timer for any duration and receive an alert when the time runs out.
- The timer can be paused and resumed as needed—ideal for tasks like cooking or exercising.

## Usage
### Digital Clock
  Upon launching the app, the Digital Clock will display the current time in a clear format. You can toggle between 12-hour and 24-hour formats in the settings.
### Setting Alarms
Navigate to the Alarm section to set, edit, or delete alarms. You can customize the alarm tone, time, and snooze options.
### Stopwatch
Access the Stopwatch feature to start, pause, and reset the stopwatch. The app also tracks lap times for more accurate time tracking.
### Timer
Use the Timer section to set a countdown for any task. When the timer runs out, you will receive an alert. You can pause or resume the timer if needed.
